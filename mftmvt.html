MFT

Multiprogramming with Fixed Tasks

MFT (Multi programming with fixed number of task) or fixed partitioning scheme: At the time of Operating System installation, MFT scheme will be partitioned the memory into fixed sized blocks.

MVT

Multiprogramming with Varialbe Tasks

MVT (Multiprogramming with a Variable number of Tasks) is the memory management technique in which each job gets just the amount of memory it needs. That is, the partitioning of memory is dynamic and changes as jobs enter and leave the system.
